# aura-guards

Layer 4 guard-chain enforcement: authorization, flow budgets, journal coupling, and leakage tracking. Pure guard logic lives under `src/guards/` with authorization bridge in `src/authorization/`.

## Tests & examples
- Tests live under `crates/aura-guards/tests/`.
- Examples live under `crates/aura-guards/examples/`.
- Run: `cargo test -p aura-guards`.
